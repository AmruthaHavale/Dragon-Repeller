# 🐉 Dragon Repeller – A Text-Based RPG Game

Welcome to **Dragon Repeller**, a simple browser-based RPG built with **HTML, CSS, and JavaScript**.  
Your mission: Defeat the dragon! Along the way, you’ll explore, new monsters, you fight them and gain your experience points and gold, using which you can buy weapons and health.

---

## ✨ Features
- 🎮 Interactive gameplay with buttons to navigate locations and actions  
- 🏪 Store system to buy health, upgrade weapons, or sell items  
- ⚔️ Multiple weapons with increasing power (stick → sword)  
- 👾 Monsters with different levels and health (Slime, Fanged Beast, Dragon)  
- 💰 Gold and XP system to track progress  
- 🩸 Health management with risk of defeat  
- 🏆 Win/lose conditions with replay option  

---

## 📂 Tech Stack
- **HTML** – Game structure and UI  
- **CSS** – Styling and layout (light/dark areas, monster stats panel)  
- **JavaScript** – Game logic, event handling, and state updates  

---

## 🚀 How to Play
1. Clone or download the repository.  
2. Open `index.html` in your browser.  
3. Use the buttons to explore the town, cave, and store.  
4. Fight monsters, collect gold, and upgrade your weapons.  
5. Defeat the dragon to win the game! 🎉
